PROJECT TITLE: Roll The Ball
PURPOSE OF PROJECT: To simulate the app Roll The Ball with similar functionality
VERSION or DATE: 1.0
HOW TO START THIS PROJECT:Run the applet called RollTheBall
AUTHORS: Chris Fall, Ryan Mikulek, Matthew MacFayden, Ronald Matos, Luis Concepcion
USER INSTRUCTIONS: 
Click once on a tile and then once on an adjacent empty space in order to move the tile.
If a piece is unable to move, it will be a different color than the rest of the pieces.
Start tiles are blue, End tiles are red, immobile tiles are magenta.
If the game is won, the path from start to end will turn green and ask you to continue or exit the game.
On completing the final level, click the 'x' in corner of the window to exit the game.
